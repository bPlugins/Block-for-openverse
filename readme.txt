Effortlessly incorporate Openverse media content into your WordPress site with this block plugin.

== Description ==

The Block for Openverse plugin seamlessly integrates Openverse media content, including audio and images, into your WordPress site. Easily access a vast library of high-quality, freely licensed media to enhance your website's visual and auditory appeal. With intuitive Gutenberg blocks, you can quickly add stunning visuals and captivating audio to your posts and pages. Unlock a world of creative possibilities and enrich your site's content with Openverse media, all within the familiar WordPress environment.


= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First install the Block For Openverse plugin
- Add the Block For Openverse block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Block For Openverse**
3. Click on the **Block For Openverse** to add the block

= Download & Upload: =
1. Download the **Block For Openverse** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`block-directory.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Block For Openverse** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
